# Manual Note To Writing Entry

## Situation

The user has messy context in Obsidian:

- notebook photos
- quick fragments
- older linked notes
- a rough idea for a writing topic

## Request

```text
Add a writing entry for this idea.
```

## What Glide Should Do

1. Read the relevant notes, links, and source context.
2. Keep Glide's working notes inside `Agent HQ/`.
3. Create or draft the requested writing note only because the user asked for it.
4. Base the first shape on the user's own notes and thoughts.
5. Add light validation when useful: related prior notes, source links, open questions, or a small research pass.
6. Preserve uncertainty instead of pretending the idea is finished.

## Output Shape

The result should be a clean linked note with:

- a concise top point
- useful source notes
- a starter shape or outline
- open questions or next research steps

The important part is continuity. Manual capture stays fast and messy; Glide helps the idea keep moving.
