# Daily Glide Check-In Prompt

Run `$glide-daily-check-in` for this vault.

Use the Glide skill as the source of truth for context loading, daily-touch selection, parallel focus scanning when supported, guardrails, and light evaluation.

Before choosing the daily touch, scan `Agent HQ/Open Loops.md` and active goals, questions, ponders, contradictions, recent decisions, and relevant area files. When several daily focus areas are plausible, explore each candidate narrowly, then integrate the findings and choose one user-facing touch.

Keep the output conversational and short. If there is no genuinely useful question, insight, advice, tracked open-loop update, or small low-risk action today, say so briefly instead of manufacturing homework.

Do not suggest financial, legal, medical, interpersonal, public, or work-sensitive actions as executable; draft or ask for approval instead.
