# Daily Glide Check-In Prompt

Run the installed `glide-daily-check-in` skill for this vault, or follow `Agent HQ/Checklists/Daily Glide Check-In.md` if the harness does not support skills.

Use Glide as the source of truth for context loading, daily-touch selection, guardrails, and light evaluation.

Before choosing the daily touch, scan `Agent HQ/Open Loops.md` and active goals, questions, ponders, contradictions, recent decisions, and relevant area files. If the harness supports parallel workers, use them for separate candidate focus areas and integrate the findings in the main thread.

Keep the output conversational and short. If there is no genuinely useful question, insight, advice, tracked open-loop update, or small low-risk action today, say so briefly instead of manufacturing homework.

Do not suggest financial, legal, medical, interpersonal, public, or work-sensitive actions as executable; draft or ask for approval instead.
