# Nightly Research Review Automation Prompt

Run `$glide-nightly-research-review` for this vault.

Suggested schedule: daily at 4am local time.

Use the Glide skill and `Agent HQ/Checklists/Nightly Research Review.md` as the source of truth.

Start with internal analysis and planning, then use parallel subagents when available for memory gaps, open loops, open research, eval/process improvement, area connections, concise representation, and stale working-memory candidates.

Keep the output internal by default. Update Agent HQ, research notes, open loops, area links, and the rolling 4-day audit when useful. Ask for approval before behavior-changing edits, destructive deletion, sensitive external actions, or changes requiring personal judgment.
