# App Interface And Computer Use

Use for connector, file, browser, or computer-use workflows.

## Interface Order

1. Use a stable connector or API.
2. Use export/import files when they are safer or easier to review.
3. Use app automation, Shortcuts, URL schemes, browser, or computer use when no better interface exists.
4. Ask before taking sensitive, irreversible, financial, public, interpersonal, or account-changing actions.

## Operating Rules

- Keep Agent HQ as the durable memory and decision layer.
- Default note updates to the Obsidian vault, not Apple Notes or other external note apps, unless the user explicitly asks to write somewhere else.
- Treat external apps as interfaces unless explicitly chosen as a source of truth.
- Start read-only and narrow.
- Record what was done, what was observed, and what still needs approval.
- Draft messages, transactions, posts, or changes before execution when stakes are meaningful.
- Prefer one app-control task per thread to avoid conflicting app state.
