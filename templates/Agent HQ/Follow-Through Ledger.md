# Follow-Through Ledger

Lightweight tracking for commitments, waiting threads, and promises that should not disappear.

This is not a task manager. Use it for things where follow-through quality matters, especially when the agent may need to notice stale loops, missed dates, or useful next steps.

## Rules

- Track only items that benefit from ongoing attention.
- Keep entries short.
- Link to the source when useful.
- Use external task apps for concrete task execution when the user uses one.
- Close, simplify, or move stale items when they stop helping.
