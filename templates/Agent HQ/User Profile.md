# User Profile

High-level context about the user for Glide to load when giving personal advice, doing research, creating decisions, or running daily check-ins.

This is not a journal. Keep it concise, useful, and easy to refresh.

## Long-Term

Stable facts, values, roles, constraints, preferences, and durable self-understanding that should shape recommendations over time.

- Add durable profile notes here.

## Lately

Recent themes, changes, stresses, opportunities, or patterns that may matter for the next few weeks or months.

- Add recent profile notes here.

## Right Now

Current priorities, active context, capacity, appetite for questions or actions, deadlines, sensitive boundaries, and near-term constraints.

- Add current profile notes here.

## Profile Rules

- Update this file when a durable signal appears in conversation, reviews, decisions, research, or daily check-ins.
- Keep `Long-Term` conservative. Change it only when stable facts or durable self-understanding change.
- Let `Lately` and `Right Now` move more often.
- Prefer short operational summaries over diary entries.
- Remove stale `Right Now` and `Lately` items when they stop being useful. Preserve durable learning elsewhere before clearing it.
- Do not copy private details from ordinary notes unless the user explicitly wants them here. Summarize only what improves future recommendations.
