# Glide Update User Profile

## Load

- `Agent HQ/AGENTS.md`
- `Agent HQ/Operating Manual.md`
- `Agent HQ/Communication Preferences.md`
- `Agent HQ/User Profile.md`
- `Agent HQ/Checklists/Update User Profile.md`
- Relevant recent Agent HQ updates, conversations, goals, decisions, ponders, questions, open loops, project links, area files, or run logs

## Process

1. Follow `Agent HQ/Checklists/Update User Profile.md`.
2. Look for durable profile signal even when there is no explicit diff.
3. Update `Agent HQ/User Profile.md` only when the change would improve future advice, decisions, research, or daily check-ins.
4. Keep `Long-Term` conservative.
5. Let `Lately` and `Right Now` move more often.
6. Prefer short operational summaries over raw notes, transcripts, or diary-like detail.
7. Remove stale recent context when it stops being useful, preserving durable learning elsewhere first when needed.

## Output

- `User Profile.md` updates when durable signal exists.
- A short note that no profile update was needed when there is no durable signal.
